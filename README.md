# Record - Audio Recording Application

A simple, elegant audio recording web application built with Next.js, React, Prisma, and PostgreSQL.

## Features

- 🎙️ Record audio directly from your browser
- ⏸️ Pause and resume recordings
- ▶️ Play back recordings
- 📥 Download recordings as WebM files
- 🗑️ Delete recordings
- 💾 Persistent storage with database
- 🌙 Dark mode support
- 📱 Responsive design

## Quick Start

### Local Development

**Option 1: Using Vercel Postgres (Recommended)**

```bash
# 1. Create a Vercel Postgres database at https://vercel.com/dashboard
# 2. Copy the connection string

# 3. Create .env.local with your DATABASE_URL
echo 'DATABASE_URL="<your-vercel-postgres-url>"' > .env.local

# 4. Install and run
npm install
npx prisma migrate deploy
npm run dev
```

**Option 2: Using Local PostgreSQL**

```bash
# 1. Create a local database
createdb record

# 2. Create .env.local
echo 'DATABASE_URL="postgresql://localhost:5432/record"' > .env.local

# 3. Install and run
npm install
npx prisma migrate deploy
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to start recording!

- **Frontend**: React 19, Next.js 16, Tailwind CSS 4

## Troubleshooting

### "Failed to fetch recordings"
- Check DATABASE_URL is set correctly in .env.local
- Verify database is accessible
- Check Vercel logs for detailed errors

### "BLOB_READ_WRITE_TOKEN not found"
- Go to Vercel project → Storage → Blob
- Create a new Blob store if needed

### "Connection refused"
- Make sure PostgreSQL is running
- Verify DATABASE_URL is correct
